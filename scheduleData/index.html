# Path URL Description
1 / Index page(This Page)
2 /routes/all Retreve All records in the Scedule
3 /roures/byrootno/< Root Number > Retreve records by specific Root Number
4 /routes/byid/< Bus ID > Retreve records by specific Bus Number
5 /roures/bystartplace/< start location > Retreve records by specific Start location
6 /routs/bydriverid/< Driver NTC Number > Retreve a record by specific Driver NTC Number
7 /routes/byconducterid/< Conductor NTC Number > Retreve a Record by specific Conductor NTC
